# FastAPI AI Post Generator

API desarrollada con FastAPI que consume APIs de inteligencia artificial para la generación de imágenes, con el objetivo de crear contenido automatizado para publicaciones en Instagram.

Actualmente, este proyecto forma parte de una arquitectura en desarrollo, donde esta API será consumida por una aplicación frontend en Angular.

## Tecnologías
- FastAPI
- PostgreSQL
- JWT Authentication
- Integración con APIs de IA

## Estado del proyecto
Proyecto en desarrollo. Algunas funcionalidades aún se encuentran en proceso de implementación.

## Cómo ejecutar

```bash
pip install -r requirements.txt
uvicorn main:app --reload
